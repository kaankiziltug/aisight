AISight Public API

Free, read-only access to AI capital expenditure data for 38+ companies. No API key required.

Base URL: https://aisight.fyi/api/v1
Free and Open
No API key, no authentication, no rate limits. Static JSON served from CDN.
Read-Only
GET requests only. No mutations, no write operations, no side effects.
Always Fresh
Updated 3x daily via automated pipeline from SEC filings and earnings data.
CORS Enabled
Access from any domain, any frontend. Build dashboards, apps, or widgets.

Endpoints

GET /api/v1/companies

Returns all 38 tracked companies with financial data including AI CapEx, revenue, market cap, year-over-year changes, and historical data.

Response Fields

Field Type Description
namestringCompany name
tickerstringStock ticker (or "PRIVATE")
aiCapexnumberAI CapEx in billions USD
totalCapexnumberTotal CapEx in billions USD
revenuenumberAnnual revenue in billions USD
marketCapnumberMarket cap in billions USD
yoyChangenumberYear-over-year AI CapEx change %
employeesnumberEmployee count
rdSpendnumberR&D spending in billions USD
categorystringCompany category
isPublicbooleanWhether publicly traded
domainstringCompany website domain
countrystringHQ country
confidenceobjectData confidence levels
historicalobjectHistorical data arrays

GET /api/v1/news

Returns latest AI spending news articles with source attribution, summaries, and related company tags.

Response Fields

Field Type Description
titlestringArticle headline
urlstringSource URL
sourcestringNews source name
datestringPublication date
summarystringBrief summary
companiesarrayRelated company names

Code Examples

const res = await fetch('https://aisight.fyi/api/v1/companies');
const data = await res.json();

// Top 5 AI spenders
const top5 = data.companies
  .sort((a, b) => b.aiCapex - a.aiCapex)
  .slice(0, 5);

top5.forEach(c => {
  console.log(`${c.name}: $${c.aiCapex}B`);
});
import requests

data = requests.get('https://aisight.fyi/api/v1/companies').json()

# Top 5 AI spenders
top5 = sorted(data['companies'], key=lambda c: c['aiCapex'], reverse=True)[:5]

for c in top5:
    print(f"{c['name']}: ${c['aiCapex']}B")
curl -s https://aisight.fyi/api/v1/companies | jq '.companies | sort_by(-.aiCapex) | .[0:5] | .[] | "\(.name): $\(.aiCapex)B"'

Security and Fair Use

Read-only
No write operations possible. All endpoints return static JSON.
Static files
No database, no server-side processing, no injection surface.

GET only
Security headers

Rate
No artificial rate limit, but data refreshes every 8 hours.
Fair use
Please cache responses locally. Do not poll more than once per hour.
Attribution
Please link back to aisight.fyi when using the data.
